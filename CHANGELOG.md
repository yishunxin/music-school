## v0.0.1 - 初始版本

### 首次发布
- 技术栈：React + Node.js + Express + SQLite
- 默认管理员账号：`admin` / `123456`

### 功能模块

#### 1. 登录认证
- JWT 令牌认证
- 登录页面（无账号密码提示）

#### 2. 账号管理
- 管理员账号增删改查

#### 3. 教师管理
- 教师信息管理
- 关联可教授的科目

#### 4. 课程类型管理
- 课程类型 = 乐器类型 + 级数（如钢琴1级、吉他2级）
- 支持设置单节课时长和单价

#### 5. 学生管理
- 学生信息管理
- 关联授课老师和课程类型（**老师为必填项**）
- 列表展示备注

#### 6. 课时管理
- **充值**：选择学生、课程类型、授课老师，输入购买课时、赠送课时、总费用；总费用自动计入收入
- **签到**：选择学生后自动关联老师和课程类型，默认消耗1课时，时间默认当前时间
- **退课**：暂不开放

#### 7. 财务管理
- 收支记录管理
- 收入/支出分类
- 报表统计

#### 8. 工资管理
- 每次课程签到直接计算一笔支出给到对应老师
- 授课老师费用 = 总费用 / 购买课时 / 2
- 月结算时根据该老师所有支出记录求和，作为月结工资

### 业务规则
- 一个学生同一时间只能有一个授课老师，只能有一种课程类型
- 一个授课老师可关联多个课程类型
- 课时充值：总课时 = 购买课时 + 赠送课时
- 课时不足提醒：剩余课时 ≤ 1 时在首页提醒
